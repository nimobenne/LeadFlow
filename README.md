# MentalCore PDF Landing Page

Conversion-focused fullstack landing page built with Next.js for selling a MentalCore YouTube channel PDF.

## Features

- Professional landing page with social proof and pricing CTA.
- Customer star reviews loaded from a backend API route.
- Lead/order intent capture form posting to backend API route.
- Ready to deploy on Vercel.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Keep defaults (`Next.js`) and deploy.
